### Basic usage

<br>

```bash
yarn start

# Expected 3000 words, 30 decks
```

<br>

### 1000 words

<br>

```bash
yarn start 1000

# Expected 1000 words, 10 decks
```

<br>

### 1000 words and shuffle

<br>

```bash
yarn start 1000 shuffle

# Expected 1000 shuffled words, 10 decks
```

<br>
<br>

### Development mode

<br>

```bash
yarn dev # ...args
```
